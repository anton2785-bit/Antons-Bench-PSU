# Antons-Bench-PSU
---
## What is this project
* This project is a PC PSU converted into a reliable power source for small projects. 
---
## Why i made this project
* I needed a bench PSU to power my small projects without the need of chaining batteries for some unstandart voltages.

## Assembly
* The PSU is oppened up and cleaned if needed.
* Then the cables are sorted in groups for the power rails.
* The wires are cut and souldered together (the access wires are cut and isolated).
* the power regulator is placed in the front plate with the banana sockets, XT-60 connectors and the USB.
* The top plate is screwed in to the main frame with heated inserts.
